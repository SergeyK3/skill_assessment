Экзамен: коммерческий директор

Сотрудник: Алексей В., коммерческий директор · Подразделение: Отдел продаж · Сессия: demo-2026-03-22

В фокусе по регламенту: навыки «Выявление потребности клиента» и «Переговоры» (1–2 навыка из чеклиста).

Часть 1 — устное интервью с LLM (регламенты, KPI)

Контекст для модели: регламент должности «Коммерческий директор», KPI отдела продаж (конверсия, маржа), фрагменты из папки инструкций (если есть).

LLM: Какой KPI отдела продаж вы считаете главным в текущем квартале и как он связан с регламентом по ведению воронки?
Алексей: Главное — конверсия из лида в сделку не ниже 25%. По регламенту каждая сделка должна иметь этапы в CRM; я требую, чтобы менеджеры не пропускали квалификацию — это напрямую влияет на конверсию.
LLM: Что вы делаете, если KPI по марже проседает при выполнении плана по объёму?
Алексей: Смотрим структуру скидок и условия; при необходимости останавливаем необоснованные уступки. В регламенте прописано согласование скидок выше порога — я это контролирую.
Оценка LLM по части 1 (условная) Согласованность ответов с регламентом: 2 / 3 (типовая ситуация)

Комментарий модели (условный): «Ответы показывают понимание KPI и связи с CRM; глубина по марже — частичная — без детализации порогов скидок из документа.»

Часть 2 — кейс от LLM (ситуационная задача)

Ситуация. Ключевой клиент требует дополнительную скидку 12% сверх утверждённой, угрожает уйти к конкуренту. Ваша маржа по сделке на грани минимума; регламент допускает скидку не более 7% без согласования с финдиректором. У вас один день до подписания.

Вопрос: Какие шаги вы предпримете и как оформите решение?
Решение Алексея (условно): Не соглашаюсь на 12% сразу; провожу созвон, уточняю объём и срок контракта; предлагаю компромисс 7% + неценовые условия. Параллельно инициирую согласование с финдиректором на исключение, если объём оправдывает отступление от маржи.
Решение принято? Да, в рамках регламента (частичное согласование)
Насколько полно 2 / 3 — учтены лимиты и эскалация; мало конкретики по доказательствам ценности для клиента

Навыки в фокусе кейса: Переговоры, Выявление потребности (через уточнение объёма/срока).

Часть 3 — оценка руководителем (генеральный директор)

По методике из PDF: «кто оценивает» — непосредственный руководитель / вышестоящий по должности.

НавыкОценка руководителя (шкала 0–3)Краткий комментарий
Выявление потребности 2 На реальных сделках видит запрос клиента; иногда уходит в «продукт» раньше времени.
Переговоры 3 Устойчив в сложных переговорах; умеет торговаться без разрушения отношений.
