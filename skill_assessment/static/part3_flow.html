Оценка навыков — рабочая сессия

Укажите client_id в URL (из /clients или /client/… → Приложения), чтобы загрузить сотрудников из ядра. Без него используется demo_client и список может быть пустым.

Оценок ещё нет — что делать: это ожидаемо в начале. В таблице нажмите Выбрать — список сократится до этого сотрудника, ниже откроется блок с кнопкой Создать сессию и старт, затем этапы 1–3. Колонки «Док. / Кейс / Рук.» и «Последняя сессия» заполнятся после прохождения. Если списка сотрудников нет — сначала добавьте их в рабочем пространстве организации.

1 Опрос по служебным документам
